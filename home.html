ChallengeAccepted

The website, built on Ruby on Rails, for people who want to spice up their live. No limits to the content of a challenge. The features include the ability to create public challenges that will be displayed on the website, as well as creating private challenges for a specific person. Ivintations are sent via email. Those who accepted the challenge can submit the proof by uploading videos and images of their accomplishments.

AlumNUTS

The app provides web development program officers an interactive snapshot of the graduated cohorts to effectively gauge the program's success during a graduate's job search and subsequent hire. Outcomes officers can log in and view targeted data from an individual graduate or from an entire cohort. The officer can view graphic analytics based on the updates a student enters and from his/her 100-Day Survey, as well as the total employment rate across all his cohorts.

EcoDash

An online discussion forum built with Sinatra and PostgreSQL, where users can create topics of their concern, comment and vote on other users’ entries. Most voted topics are displayed first. Users have the ability to add tags to the topic they’re creating and search entries by tags. The first tag displayed is the user location, obtained via an API call. The app includes full user authentication. Accounts are password protected.

SimonSays

A childhood game, built with HTML, CSS, JavaScript and JQuery. It generates a growing sequence of colors that the player has to repeat. Press Start to start a game. You will see a sequence of blinking colors. Try to memorize and repeat it by pressing the matching buttons to advance to the next level. The number of blinking colors will increase with each round. If the pressed sequence is not correct, a window will pop up and show a number of completed levels.
